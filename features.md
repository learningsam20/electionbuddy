# ElectionBuddy Feature Catalog

## 🗳️ Citizen Features
| Feature | Tech | Description |
| :--- | :--- | :--- |
| **Maturity Quiz** | React + FastAPI | Assess knowledge of democratic processes with score tracking. |
| **Social Generator** | Gemini 1.5 Flash | Generate viral, neutral voting awareness posts in multiple languages. |
| **Manifesto Summarizer** | Gemini 1.5 Flash | Summarize complex candidate promises into 3-4 bullet points. |
| **Voter Issue Hub** | STT + FastAPI | Submit anonymous text/voice issues to a constituency heatmap. |
| **Booth Locator** | Google Maps API | Find nearest polling booth with real-time distance and wait times. |
| **Chatbot** | Gemini 1.5 Flash | AI assistant to answer FAQ about voting rules and IDs. |

## 👔 Candidate Features
| Feature | Tech | Description |
| :--- | :--- | :--- |
| **Campaign Assistant** | Gemini 1.5 Flash | AI-assisted drafting of speeches, press releases, and posts. |
| **Issue Heatmap** | Recharts + SQL | Visualize top concerns of citizens in your specific constituency. |
| **Sentiment Tracking** | Gemini (NLP) | Analyze media sentiment from news URLs. |
| **Master Profile** | React Forms | Single source of truth for your digital campaign presence. |

## 🛡️ Officer Features
| Feature | Tech | Description |
| :--- | :--- | :--- |
| **AI Content Triage** | Gemini (Moderation) | Automatic scanning of candidate media for policy violations. |
| **Resource Map** | Google Maps API | Track security personnel and readiness across all polling booths. |
| **Multi-lingual Alerts** | Gemini (Translate) | Push translated notifications to the entire constituency instantly. |

## ⚙️ Admin Features
| Feature | Tech | Description |
| :--- | :--- | :--- |
| **Cloud Usage** | SQL + Recharts | Detailed breakdown of Gemini token usage and API costs. |
| **Audit Logs** | Immutable SQL | Track every administrative and moderation action for transparency. |
| **Anomaly Detection** | Python | Monitor for suspicious registration spikes or API abuse. |
